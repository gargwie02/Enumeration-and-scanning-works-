# Enumeration-and-scanning-works-
## Enumeration Lab

### Objective

To gather detailed information about the target system using enumeration techniques.

### Tools Used

* Nmap
* smbclient
* enum4linux
* theHarvester
* Nessus

### Procedure

1. Performed Nmap scan to identify open ports and services
2. Identified SMB service on port 445
3. Used smbclient to list shared folders
4. Used enum4linux to enumerate users and groups
5. Used theHarvester to collect email addresses
6. Performed vulnerability scanning using Nessus

### Result

Successfully identified:

* Open ports
* SMB shares
* Users and groups
* Email addresses
* Vulnerabilities

### Conclusion

Enumeration helps in gathering detailed information about a target system, which is essential for further exploitation.
